== 创建一个 Raspberry Pi 账户

要使用Raspberry Pi服务，您必须首先创建一个 https://id.raspberrypi.com[Raspberry Pi ID]。

在浏览器中，导航到 https://id.raspberrypi.com[id.raspberrypi.com]。

输入电子邮件和密码。单击 **注册(Sign up)** 按钮创建您的帐户。

image::images/create_account_identity.png[width="80%"]

创建帐户后，您将收到一封验证电子邮件。

image::images/verify_identity.png[width="80%"]

单击电子邮件中的 **验证电子邮件(Verify email)** 按钮以验证您的电子邮件地址并完成帐户创建。

image::images/signed_in_identity.png[width="80%"]

创建Raspberry Pi ID后，您可以使用 **使用Raspberry Pi ID登录(Sign in with Raspberry Pi ID)** 按钮登录Raspberry Pi服务。
