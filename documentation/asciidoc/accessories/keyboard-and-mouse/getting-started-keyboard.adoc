== 键盘入门

我们的官方键盘包括三个主机USB端口，用于连接外部设备，例如USB鼠标、USB驱动器和其他USB控制的设备。

该产品的微型USB端口用于连接到Raspberry Pi。通过键盘内置的USB集线器，Raspberry Pi控制三个USB Type A端口并为其供电。

image::images/back-of-keyboard.png[width="80%"]

=== 键盘功能

Raspberry Pi键盘有三个锁定键：`Num Lock`、`Caps Lock` 和 `Scroll Lock`。右上角有三个LED指示启用了哪些锁。

image::images/num-cap-scroll.png[width="80%"]

`Num Lock`:: 允许使用字母键上的红色数字键，有效地创建数字小键盘。按 `Num Lock` 键启用和禁用此模式。

`Caps Lock`:: 允许键入大写字母；在此模式下按 `Shift` 键键入小写字母。按 `Caps Lock` 键启用和禁用此模式。

`Scroll Lock (ScrLk)`:: 允许在没有鼠标的情况下使用光标键浏览网页和电子表格。按住Fn键时按 `ScrLk` 键启用和禁用此模式。
