== 简介

https://www.raspberrypi.com/products/sense-hat/[Raspberry Pi Sense HAT] 是一款附加板，可为您的Raspberry Pi提供一系列传感功能。板载传感器允许您监控压力、湿度、温度、颜色、方向和运动。8×8 RGB LED矩阵允许您可视化来自传感器的数据。五按钮操纵杆允许用户与您的项目进行交互。

image::images/Sense-HAT.jpg[width="70%"]

Sense HAT 最初是为在国际空间站上使用而开发的，是 https://raspberrypi.org[树莓派基金会] 与 https://www.esa.int/[欧洲航天局] 合作开展的教育 https://astro-pi.org/[Astro Pi] 计划的一部分。它可以用于任何需要位置、运动、方向或环境传感的项目。

官方支持的 xref:sense-hat.adoc#use-the-sense-hat-with-python[Python library] 提供对板载传感器、LED矩阵和操纵杆的访问。Sense HAT与任何具有40针GPIO接口的Raspberry Pi设备兼容。
